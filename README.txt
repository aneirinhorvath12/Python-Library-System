In this program all user inputs and outputs can be done in the Tkinter GUI
To use the system enter the necessary data for any task:
 -- Book Name for searching books
 -- Book ID and User ID for checking out Books
 -- Book ID for returning books
 -- Book Name for weeding books
and then press the button of the task they would like to perform., the program will always either return something to the user 
or confirm that the task is complete.
To weed the book library the user should input the name of the book they want and the program will return the number of times
that title has been checked out since January 1st 2020, the user can then decide with this information which books they wish to keep.
